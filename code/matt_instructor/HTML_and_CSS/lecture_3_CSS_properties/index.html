CSS Properties

Click here to see the documentation that also includes CSS Lengths and CSS Colors

colors, you can change the color


font-family, you can change the font to something else, such as Courier New for example


font-size, change the font size to 2 rem


text-align, makes the font be positioned on the left, center or right


background-color, make the background a different color, do it by color name, jex or rgb


background-image, gives your content a backgrounds image: make sure to set the size


margin, set the margin, the space between elements


padding, Set the padding, the space inside an element


border, Set the border


width, Set the width of the element (only works on block elements)


height, Set the height of the element (only works on block elements)
